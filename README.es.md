# ⚖️ Generador de Contratos Legales

> 🌐 [English](README.md) · [Français](README.fr.md) · [Português](README.pt.md) · [中文](README.zh.md) · [日本語](README.ja.md)

Una aplicación web gratuita y de código abierto que genera contratos legales personalizables a partir de plantillas profesionales. Exporta a **Markdown** o **PDF** — completamente del lado del cliente, sin servidor necesario.

> ⚠️ **Aviso**: Esta herramienta genera plantillas de contratos solo con fines informativos. **No** proporciona asesoría legal. Siempre consulta a un abogado calificado antes de firmar cualquier documento legal.

## ✨ Características

- 📝 **10+ Plantillas de Contratos** — NDAs, servicios freelance, ventas, arrendamientos, empleo, préstamos y más
- 🌍 **6 Idiomas** — Inglés, Español, Francés, Portugués, Chino, Japonés
- 📄 **Exportación PDF y Markdown** — Genera documentos profesionales del lado del cliente
- ✏️ **Editor de Contratos Personalizado** — Escribe tus propios contratos con barra de herramientas y biblioteca de cláusulas
- 🔒 **100% Del Lado del Cliente** — Ningún dato sale de tu navegador
- 📱 **Diseño Responsivo** — Funciona en escritorio, tablet y móvil
- 🎨 **UI Premium Oscura** — Diseño moderno con glassmorfismo

## 🚀 Inicio Rápido

1. Clona el repositorio:
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. Abre `index.html` en tu navegador — ¡eso es todo!

O despliega en **GitHub Pages** para hosting gratuito.
