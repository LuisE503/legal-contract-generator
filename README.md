# ⚖️ Legal Contract Generator

> 🌐 [Español](README.es.md) · [Français](README.fr.md) · [Português](README.pt.md) · [中文](README.zh.md) · [日本語](README.ja.md)

A free, open-source web application that generates customizable legal contracts from professional templates. Export to **Markdown** or **PDF** — entirely client-side, no server required.

> ⚠️ **Disclaimer**: This tool generates contract templates for informational purposes only. It does **not** provide legal advice. Always consult a qualified attorney before signing any legal document.

## ✨ Features

- 📝 **10+ Contract Templates** — NDAs, freelance services, sales, leases, employment, loans, and more
- 🌍 **6 Languages** — English, Spanish, French, Portuguese, Chinese, Japanese
- 📄 **PDF & Markdown Export** — Generate professional documents client-side
- ✏️ **Custom Contract Editor** — Write your own contracts with a rich toolbar and clause library
- 🔒 **100% Client-Side** — No data leaves your browser
- 📱 **Responsive Design** — Works on desktop, tablet, and mobile
- 🎨 **Dark Premium UI** — Modern glassmorphism design

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. Open `index.html` in your browser — that's it!

Or deploy to **GitHub Pages** for free hosting.

## 🛠️ Tech Stack

- **HTML5** + **CSS3** + **Vanilla JavaScript**
- **html2pdf.js** for PDF generation
- No frameworks, no build tools, no dependencies to install
