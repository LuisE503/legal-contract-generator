# ⚖️ 法律合同生成器

> 🌐 [English](README.md) · [Español](README.es.md) · [Français](README.fr.md) · [Português](README.pt.md) · [日本語](README.ja.md)

一个免费的开源网络应用程序，可从专业模板生成可自定义的法律合同。导出为 **Markdown** 或 **PDF** — 完全在客户端运行，无需服务器。

> ⚠️ **免责声明**：此工具仅出于信息目的生成合同模板。它**不**提供法律建议。在签署任何法律文件之前，请务必咨询合格的律师。

## ✨ 功能特点

- 📝 **10+ 合同模板** — 保密协议、自由职业服务、销售、租赁、雇佣、贷款等
- 🌍 **6 种语言** — 英语、西班牙语、法语、葡萄牙语、中文、日语
- 📄 **PDF 和 Markdown 导出** — 在客户端生成专业文档
- ✏️ **自定义合同编辑器** — 使用工具栏和条款库编写自己的合同
- 🔒 **100% 客户端** — 数据不会离开您的浏览器
- 📱 **响应式设计** — 适用于桌面、平板和手机

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. 在浏览器中打开 `index.html` — 就这样！
