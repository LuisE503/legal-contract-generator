# ⚖️ Gerador de Contratos Legais

> 🌐 [English](README.md) · [Español](README.es.md) · [Français](README.fr.md) · [中文](README.zh.md) · [日本語](README.ja.md)

Um aplicativo web gratuito e de código aberto que gera contratos legais personalizáveis a partir de modelos profissionais. Exporte para **Markdown** ou **PDF** — inteiramente no lado do cliente, sem necessidade de servidor.

> ⚠️ **Aviso**: Esta ferramenta gera modelos de contratos apenas para fins informativos. **Não** fornece aconselhamento jurídico. Sempre consulte um advogado qualificado antes de assinar qualquer documento legal.

## ✨ Funcionalidades

- 📝 **10+ Modelos de Contratos** — NDAs, serviços freelance, vendas, aluguéis, emprego, empréstimos e mais
- 🌍 **6 Idiomas** — Inglês, Espanhol, Francês, Português, Chinês, Japonês
- 📄 **Exportação PDF e Markdown** — Gere documentos profissionais no lado do cliente
- ✏️ **Editor de Contratos Personalizado** — Escreva seus próprios contratos com barra de ferramentas e biblioteca de cláusulas
- 🔒 **100% No Lado do Cliente** — Nenhum dado sai do seu navegador
- 📱 **Design Responsivo** — Funciona em desktop, tablet e celular

## 🚀 Início Rápido

1. Clone o repositório:
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. Abra `index.html` no seu navegador — é isso!
