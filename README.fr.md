# ⚖️ Générateur de Contrats Juridiques

> 🌐 [English](README.md) · [Español](README.es.md) · [Português](README.pt.md) · [中文](README.zh.md) · [日本語](README.ja.md)

Une application web gratuite et open-source qui génère des contrats juridiques personnalisables à partir de modèles professionnels. Exportez en **Markdown** ou **PDF** — entièrement côté client, aucun serveur requis.

> ⚠️ **Avertissement** : Cet outil génère des modèles de contrats à titre informatif uniquement. Il ne fournit **pas** de conseils juridiques. Consultez toujours un avocat qualifié avant de signer tout document juridique.

## ✨ Fonctionnalités

- 📝 **10+ Modèles de Contrats** — NDAs, services freelance, ventes, baux, emploi, prêts et plus
- 🌍 **6 Langues** — Anglais, Espagnol, Français, Portugais, Chinois, Japonais
- 📄 **Export PDF et Markdown** — Générez des documents professionnels côté client
- ✏️ **Éditeur de Contrats Personnalisé** — Rédigez vos propres contrats avec barre d'outils et bibliothèque de clauses
- 🔒 **100% Côté Client** — Aucune donnée ne quitte votre navigateur
- 📱 **Design Responsive** — Fonctionne sur desktop, tablette et mobile
- 🎨 **UI Premium Sombre** — Design moderne avec glassmorphisme

## 🚀 Démarrage Rapide

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. Ouvrez `index.html` dans votre navigateur — c'est tout !
