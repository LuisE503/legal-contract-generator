# ⚖️ 法的契約書ジェネレーター

> 🌐 [English](README.md) · [Español](README.es.md) · [Français](README.fr.md) · [Português](README.pt.md) · [中文](README.zh.md)

プロフェッショナルなテンプレートからカスタマイズ可能な法的契約書を生成する、無料のオープンソースWebアプリケーション。**Markdown** または **PDF** にエクスポート — 完全にクライアントサイドで動作、サーバー不要。

> ⚠️ **免責事項**：このツールは情報提供のみを目的として契約書テンプレートを生成します。法的助言を提供するものでは**ありません**。法的文書に署名する前に、必ず資格のある弁護士にご相談ください。

## ✨ 機能

- 📝 **10以上の契約書テンプレート** — 秘密保持契約、フリーランスサービス、売買、賃貸、雇用、ローンなど
- 🌍 **6言語対応** — 英語、スペイン語、フランス語、ポルトガル語、中国語、日本語
- 📄 **PDFとMarkdownエクスポート** — クライアントサイドでプロフェッショナルな文書を生成
- ✏️ **カスタム契約書エディター** — ツールバーと条項ライブラリで独自の契約書を作成
- 🔒 **100%クライアントサイド** — データはブラウザから外に出ません
- 📱 **レスポンシブデザイン** — デスクトップ、タブレット、モバイルに対応

## 🚀 クイックスタート

1. リポジトリをクローン：
   ```bash
   git clone https://github.com/YOUR_USERNAME/legal-contract-generator.git
   ```
2. ブラウザで `index.html` を開く — それだけです！
